IMPORTANT: Add your resume here!

Please add your resume file as "resume.pdf" in this public folder.

The download button in your portfolio will link to: /resume.pdf

To add your resume:
1. Name your resume file as "resume.pdf"
2. Place it in this public folder (frontend/public/)
3. The download button will automatically work

Alternative formats:
- If you want to use a different name, update the href in App.jsx line ~70
- You can also use .doc, .docx formats but PDF is recommended
